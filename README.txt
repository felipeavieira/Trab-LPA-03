1. Ambiente de trabalho e conteúdo

1.1. Os códigos fonte dos programas em C (.c) foram escritos e compilados no ambiente de desenvolvimento 
integrado (IDE) Code::Blocks, versão 13.12 rev 9501.
A arquitetura do sistema foi o windows 10 x64.

1.2. O compilador dos executáveis (.exe) foi o mingw32-gcc.exe, gcc 4.7.1 Windows/unicode - 32bit.

1.3. Foram utilizados os seguintes parâmetros de compilação:

Linker settings: -static-libgcc -static-libstdc++

1.4. Nos programas foram usadas duas bibliotecas: stdio.h e stdlib.
As bibliotecas estão localizadas na pasta "Bibliotecas (.h)", no repositório do trabalho.

1.5. Um exemplo de entrada foi dado, e está localizado no repositório, com o nome "Entrada-exemplo".

1.6. O código fonte (.c) e executável (.exe) estão no repositório do trabalho.

2. Programa no repositório 

2.1  Matriz esparsa

Esse programa permite o inserção e manipulação de uma matriz esparsa. Estão disponíveis as seguintes operações:
- Criação
- Remoção
- Adicionar valores
- Consultar posições
- Somar linhas
- Somar colunas
